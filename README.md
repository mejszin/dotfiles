# Installation

```
sudo apt install neovim
```

```
mkdir ~/.config/nvim
touch ~/.config/nvim/init.vim
touch ~/.config/nvim/plugins.vim
```

```
curl -fLo ~/.config/nvim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

# vim-plug

```
:PlugStatus
:plugInstall
:PlugUpdate
:PlugDiff
:PlugClean
:PlugUpgrade
```
